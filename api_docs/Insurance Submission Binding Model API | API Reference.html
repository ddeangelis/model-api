Get number of inferences calculated

GET /meter_count

Get the number of inferences calculated with the model and delivered via API, given start and end dates and times.

startDate

Start date-time for the meter count, in UTC time, in RFC3339 format. Example: "1985-04-12T23:20:50.52Z"

type
string (date-time)
in
query
endDate

End date-time for the meter count, in UTC time, in RFC3339 format. Example: "1985-05-12T23:20:50.52Z"

type
string (date-time)
in
query
200 OK

Integer count of the number of inferences calculated and delivered between the start date/time and the end date/time.

401 Unauthorized

Invalid or missing API key.

Response Example (200 OK) 
212

Model

Get probability of binding

POST /model/binding_probability

Calculate and return the estimated probability of binding a submission representing new business.

Submission

JSON object representing all known information at the time of submission for new business. All fields are required, though "Product" can take the value "NA".

Request Example
{
  "SubmissionNumber": "string",
  "CompanyCode": "string",
  "DivisionCode": "string",
  "InsuredName": "string",
  "InsuredAddress": "string",
  "InsuredCity": "string",
  "InsuredState": "string",
  "InsuredZip": "string",
  "GroupLine": "string",
  "Product": "string"
}
200 OK

Object contain a probability of binding (0-1), percentile of binding probability relative to the training set, and an md5sum of the submission details for data integrity verification.

400 Bad Request

Error: Missing or malformed submission data. A submission number is required.

401 Unauthorized

Invalid or missing API key.

500 Internal Server Error

Warning: Unexpected submission data. Model performance may degrade.

Response Example (200 OK) 
{
  "estimated probability": 0.35,
  "estimated percentile": 80.55,
  "md5sum": "93d93f3c7089359efd66b9cf05279574"
}
Response Example (500 Internal Server Error) 
{
  "estimated probability": 0.35,
  "estimated percentile": 80.55,
  "md5sum": "93d93f3c7089359efd66b9cf05279574"
}
